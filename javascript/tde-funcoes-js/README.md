## Funções Criadas

| Tipo | Nome | Uso |
|------|------|-----|
| Nomeada | `somar`, `subtrair`, `multiplicar`, `dividir` | Executam operações básicas. |
| Anônima | `filtrarPares` | Filtra números pares de um array. |
| Arrow Function | `calcularMedia` | Calcula a média dos números. |
| Callback | `executarOperacao` | Recebe outra função e executa. |
| DOM/Eventos | `addEventListener()` | Mostra tudo na tela sem console. |
